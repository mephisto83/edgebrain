# EdgeBrain

Federated machine learning platform for IoT edge devices with privacy-preserving training.

## Features
- Privacy-preserving federated training across edge devices
- Differential privacy budget management
- Device fleet management and monitoring
- Automatic model aggregation strategies
- Edge-optimized model deployment
- React + Tailwind SaaS webapp with Firebase Auth

## Quick Start
1. Clone this repo
2. Install: `pip install -e .`
3. Run: `edgebrain --help`

## Web App
The `webapp/` directory contains a full SaaS frontend built with React + Tailwind CSS.
